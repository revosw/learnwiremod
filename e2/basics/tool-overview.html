Finding it in the menu

Open up the spawn menu with Q, and have a look at the Wire tab. Under Chips, Gates you'll find Expression 2.

You can create, open, copy, rename and delete files. Folders are different, you can create them but not rename, nor delete them. Issue on Github

Editor overview

When you select the tool and right click outside of the spawn menu, you'll open the editor. Optionally, you can click the button on the bottom of the tool's spawn menu and open the editor there, but I have personally never used that button.

Are you ready to learn about Expression 2?
@name Hello World
@inputs A B
@outputs Add Sub Mul Div
@outputs GreaterThan Highest Lowest Overflow Overflow Overflow Overflow
@outputs Vector:vector
@persist D
@trigger all

Add = A + B
Sub = A - B
Mul = A * B
Div = A / B

GreaterThan = A > B

if(A > B) {
Highest = A, Lowest = B
} else {
Highest = B, Lowest = A
}

Vector = vec(A, B, 0)
Vector = Vector + vec(0, 0, A + B)
Vector = Vector:normalized()
