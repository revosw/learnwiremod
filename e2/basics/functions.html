Rules of a function

Functions are built a certain way, and you have to follow these rules. A function has three parts:

  • Function name
  • Parentheses
  • Parameters (Optional)
functionName("Parameter 1", "Parameter 2", "Parameter 3", "...")

Function name comes first. It is case-sensitive, so fuNCTiONnAmE() is different from funtionName()
