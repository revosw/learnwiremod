Variables

    The one true storage solution

    Variables exist because we need a way to store values. You write one by starting the word with an uppercase letter: 

    Number = 1
Calculation = 5 * 5
NaN = sqrt(-1)

    Variables can be redefined - that is the reason they're called variables:

    Number = 1
print(Number) #prints 1

Number = 2
print(Number) #prints 2
