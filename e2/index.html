I welcome YOU...

...to the course that will take you from absolute zero programming experience to an independent Expression 2 developer. Before you rush out in excitement to your favorite section, let me explain what purpose this course serves.

This course is a resource that is partly progression-agnostic. Even if you aren't new, I strongly recommend that you at least go through every single part of The Basics™ before jumping into the other sections like EGP, holograms, effects and such. You never know, maybe you learn something new?

This course is NOT a replacement for all other documentation. You won't find every single function explanation here, those are found in the E2Helper inside the editor.

This course is NOT going to cover the Lua behind how Expression 2 works

@name Hello World
@inputs A B
@outputs Add Sub Mul Div
@outputs GreaterThan Highest Lowest Overflow Overflow Overflow Overflow
@outputs Vector:vector
@persist D
@trigger all

Add = A + B
Sub = A - B
Mul = A * B
Div = A / B

GreaterThan = A > B

if(A > B) {
Highest = A, Lowest = B
} else {
Highest = B, Lowest = A
}

Vector = vec(A, B, 0)
Vector = Vector + vec(0, 0, A + B)
Vector = Vector:normalized()
