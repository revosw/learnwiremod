Introduction

About the course

I welcome you to my tutorial series for all wiremod related. Here we will be exploring Expression 2 in all its glory.

@name Hello World
@inputs A B
@outputs Add Sub Mul Div
@outputs GreaterThan Highest Lowest
@outputs Vector:vector
@persist D
@trigger all

Add = A + B
Sub = A - B
Mul = A * B
Div = A / B

GreaterThan = A > B

if(A > B) {
    Highest = A, Lowest = B
} else {
    Highest = B, Lowest = A
}

Vector = vec(A, B, 0)
Vector = Vector + vec(0, 0, A + B)
Vector = Vector:normalized()

About the presenter

I am Revolution, some half bunnyhopper half coder who just enjoyed wiremod so much that he decided to make a whole website dedicated for teaching it. For long I wanted to get really good, but no expression 2 youtube tutorial series could really deliver to my expectations. I needed a tutorial series that could take me from zero to hero without the need for searching up the terminology for everything because it wasn't thoroughly explained, and without having to jump from series to series.
