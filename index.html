How to install Wiremod

  • From the workshop (recommended)
  • From SVN - Windows 10/8/7

The most convenient method is to just install the Wiremod addon from the workshop by hitting the subscribe button, and you're done.

Setting up the repository

For those of you who want the bleeding edge builds of Wiremod, this is for you. The workshop addon isn't updated that frequently. To proceed, we first need to determine if your windows is 32-bit or 64-bit. Open the start menu and type "system information". Open it up, then check the row where it says "System type". It should say "x86-based" or "x64-based". x86 is 32-bit, x64 is 64-bit.

Download and install TortoiseSVN (this is where 32-bit and 64-bit comes in)

Navigate to your steam folder, and proceed to \steamapps\common\GarrysMod\garrysmod\addons

Inside the addons folder, open the context menu (right click). You now have an option called "SVN Checkout..."

Paste the link "https://github.com/wiremod/wire/trunk" into URL of repository and click OK

To update the wire repository, just right click the folder and click "SVN Update".

But having to manually update it every time is tedious... Why not automate it?

Automating repository update

Open the start menu, and start up Powershell ISE

In the white box on top, paste the two lines:
cd C:\"Program Files"\TortoiseSVN\bin
TortoiseProc.exe -command "update" -path "<path to wire folder>" -closeonend 1

Replace "path to wire folder" with the actual path to your wire folder inside the addons-folder. It should now look something like this:

Click File > Save As...

Choose a location to your liking. I saved mine inside a folder called "scripts" in Documents. You can call the script "updatewire"

Test to see if the script actually works by right clicking the file and choose Run with PowerShell. It won't work if either the path to TortoiseSVN, or the path to the wire folder is incorrect.

After saving the script, open Task Scheduler (type taskschd.msc if you can't find it) from the start menu. In there, create a basic task like this:

You can name the task Update wire repository, and click next

When I log on, next

Start a program, next

Write "powershell.exe" in the "Program/script" box.

Fill the "Add arguments" box with: -ExecutionPolicy Bypass C:\Users\<user>\Documents\scripts\updatewire.ps1

Don't forget to change <user> with your actual windows username

Finish up by pressing next, then finish.

We need to test our task. Click on Task Scheduler Library, and scroll down the list to find your task. If it's not there, hit F5 to refresh. Right click the task, and press Run.

You should first see a powershell window in a split second, then the TortoiseSVN updating the repository. If so, CONGRATULATIONS! You don't need to think of ever updating the wire repository on your computer.

About Wiremod For All

What?

Wiremod For All is a website that aims to guide you through all the different parts of Wiremod.

Why?

The reason I launched this site is because most parts of Wiremod has had a steep learning curve for years. I want to change this, and make everyone, even the ones not familiar with Wiremod, able to learn the ins and outs of everything in Wiremod.

How?

This site is hosted via Github Pages. It uses the Materializecss front-end framework, and a custom build of highlight.js syntax highlighting.

Who?

You can find me on Steam and Discord (Revolution#7018).
