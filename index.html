About Wiremod For All

What?

Wiremod For All is a website that aims to guide you through all the different parts of Wiremod.

Why?

The reason I launched this site is because most parts of Wiremod have been black boxes with little to no information as to how to use things for years. I want to change this, and make everyone familiar with everything in Wiremod.

How?

This site is hosted via Github Pages. It uses the Materializecss design framework, and highlight.js syntax highlighting.

Who?

You can find me on Steam.
